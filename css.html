What is CSS?

CSS is Cascading Style Sheet in full.

CSS is a way that helps a web designer to modify a website in a way that the end user would want.

Different ways how CSS can be used in a web page.

  • Inline CSS - This is used in the html file in a specific tag that the designer is targeting. This is done with the styke attribute.
  • Internal CSS - This is also in the html file but this is written in the head tag of the page using the style tag with different selectors.
  • External CSS - This is an external file with the .css extension that contains all the different styles for all the elements. The external css is just linked to the html page that the styles have to apply to.

CSS selectors

  • Element tags - These are the tags of the different elements that one wants to style.
  • Classes - Here the syntax for writitng a class in css is by using the . symbol.
  • Ids - To identify an id the, CSS checks for the # during the styling process
  • Note ids take more priority than classes and classes take more priority than element tags.
